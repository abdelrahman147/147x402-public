Authentication & Payment

All requests are paid via the X402 protocol with automatic processing on Base or Solana. Price: $0.0004/request.

Errors

  • 400 – Bad Request
  • 429 – Rate Limit Exceeded (retryAfter)
  • 500 – Internal Server Error
