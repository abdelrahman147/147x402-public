147x402 API

Base URL

https://147.402

Payment

  • Protocol: X402
  • Networks: Solana, Base
  • Price: $0.0004/request
  • Facilitator: https://facilitator.payai.network

Prefixes

  • / — Solana (default)
  • /solana/* — force Solana
  • /base/* — force Base

Endpoints

GET /token/<address>

Detailed token info.

curl "https://147.402/token/<address>"

POST /balances

Wallet balances across networks.

curl -X POST "https://147.402/balances" \
-H "Content-Type: application/json" \
-d '{"walletAddress":"0x742d35...","networks":[1,56,137]}'

GET /networks

curl "https://147.402/networks"

GET /tokens/search

curl "https://147.402/tokens/search?q=bitcoin&limit=10"

GET /bars

curl "https://147.402/bars?symbol=BTC&from=1640995200&to=1641081600&resolution=1D"
